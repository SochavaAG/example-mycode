flex-start | center | flex-end | stretch (default)

Flex контейнер
Выравнивание колонок сетки внутри блока сетки
justify-content
flex-start (default) | flex-end | center | space-beetween | space-around | space-evenly
Flex контейнер
Используется для выравнивания элементов по главной оси – направлению, в котором flex-direction задает поток
order
число 0 (default)
Flex элемент
Устанавливает порядок расположения элемента в flex или grid контейнере
flex-grow
число 0 (default)
Flex элемент
Устанавливает коэффициент увеличения гибкости, который определяет, какая часть оставшегося пространства гибкого контейнера должна быть назначена основному размеру гибкого элемента
flex-shrink
число 1 (default)
Flex элемент
Устанавливает коэффициент сжатия гибкого элемента. Если размер всех гибких элементов больше гибкого контейнера, элементы сжимаются в соответствии с flex-shrink
flex-basis
< размер контейнера > | auto (default)
Flex элемент
Устанавливает начальный основной размер гибкого элемента. Он устанавливает размер поля содержимого, если иное не установлено с помощью параметра box-sizing
flex
none | [<'flex-grow'> <'flex-shrink'>? || <'flex-basis'>]
Flex элемент
Сокращенное свойство CSS flex определяет, как будет увеличиваться или уменьшаться гибкий элемент, чтобы соответствовать доступному пространству в его гибком контейнере
align-self
start | center | end | stretch (default)
Flex элемент
Выравнивает flex-элементы по текущей flex-линии, переопределяя значение свойства align-items. Если у какого-либо flex-элемента margin в поперечной оси выставлен в auto, то align-self игнорируется

(min-width | max-width) > flex-basis > width

для свойства CSS flex-direction: row:

приоритет min-width | max-width вышем чем у flex-basis

приоритет flex-basis вышем чем у width

(min-height | max-height) > flex-basis > height

для свойства CSS flex-direction: column:

приоритет min-height | max-height вышем чем у flex-basis

приоритет flex-basis вышем чем у height
